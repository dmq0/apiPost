# 命令行中的apiPost工具🎉
## 亮点💫
如果你不希望下载apipost进行数据的请求，而是希望通过命令行方便快捷的请求数据，那么你可以试试@dmqcli/api
便捷轻量，显示简约清晰。
# 安装
```bash
npm i -g @dmqcli/api
```
# 使用
### 支持多种方法
```bash
api
```
### 快速进行get请求
```bash
api get
```
get请求参数请写在地址中
技巧：命令行中鼠标右键即可粘贴，不要忘记了~
### 快速进行post请求
```bash
api post
```
post请求参数如果没有略过弹出框即可，如果有请使用逗号进行分隔。
# 其他
除了这两种请求之外还支持其他多种方法，快来尝试吧！
